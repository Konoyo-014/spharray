# Tests

The test suite is organized by layer:

- `core/`: numerical and signal-processing fundamentals.
- `repro/`: reproducibility-facing wrappers and reference-root behavior.
- `plotting/`: plotting helpers that should run without GUI interaction.
- `experimental/`: optional FOA experimental modules.

Rafaely plotting regression tests were moved out to:

`../_rafaely_plot_bundle/rafaely_plotting/tests`
